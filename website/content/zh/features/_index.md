# 核心特性

## 🚀 高性能架构

### 智能缓存
- **LRU 缓存**：基于最近最少使用策略自动管理热点数据
- **TTL 支持**：可配置缓存过期时间
- **缓存预热**：自动缓存高频访问数据

### 优化设计
- **JSON 编码池**：降低内存分配，提升编码效率
- **响应大小控制**：智能限制响应体，降低上下文溢出风险
- **并发优化**：支持高并发请求处理

## 🔒 安全与可靠

### 多认证模式
- **API Key 认证**：简单高效，适合服务到服务调用
- **Bearer Token 认证**：适合统一鉴权链路
- **Basic 认证**：适用于受控内网环境

### 密钥管理
- **安全存储**：支持内存存储与过期控制
- **密钥轮换**：支持 API Key 与 Token 轮换
- **随机生成**：内置高强度凭据生成能力
- **环境变量接入**：可通过环境变量安全注入

### 输入清理
- **危险字符过滤**：降低 SQL/XSS/命令注入风险
- **URL 校验**：仅允许受控协议
- **长度限制**：避免超长输入导致异常负载

## 📊 完整云原生服务集成

### Kubernetes 管理
- 28 个工具覆盖核心资源管理
- Pod/Deployment/Service/Namespace/Node 全流程操作

### 应用发布
- 31 个 Helm 工具
- Chart 管理、安装升级、仓库维护

### 监控与可观测性
- **Prometheus**：20 个工具，指标采集与查询
- **Grafana**：36 个工具，仪表盘与可视化
- **Jaeger**：8 个工具，分布式追踪
- **OpenTelemetry**：9 个工具，统一观测链路

### 日志与检索
- **Elasticsearch**：14 个工具，日志索引与搜索
- **Kibana**：52 个工具，日志分析与可视化

### 告警管理
- **Alertmanager**：15 个工具
- 告警规则、路由、静默与通知策略

## 🔧 灵活运行模式

### SSE 模式
- 实时事件推送
- 低延迟交互
- 适合监控与流式场景

### Streamable-HTTP 模式
- 现代 MCP 传输方式
- 同一端点支持请求/响应与流式能力
- 生产环境推荐优先使用
